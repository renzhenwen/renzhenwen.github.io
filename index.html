Research Interests


  • Image set classification, i.e., construct an image set which consists of a set of images, and then measure the distance between two sets.
  • Feature extraction and feature selection, i.e., extract or select the important features from high-dimensional data for addressing the curse of dimensionality.
  • Multiple kernel learning, i.e., learn an optimal consensus kernel from a candidate kernel pool.
  • Mult-view clustering, i.e., integrate the supplementary and consistent information of multiple views for clustering tasks.
  • Deep learning, i.e., use deep learning for clustering and classification tasks.
